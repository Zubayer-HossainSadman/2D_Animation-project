A beautiful, retro-style 2D animation built with **Python + GLFW + PyOpenGL** (immediate mode) featuring:

- Heavy **red rain** that falls endlessly  
- Realistic **wind effect** (left/right arrows to blow rain sideways)  
- Adjustable **wind strength** (W/S keys)  
- Smooth **day → night cycle** (Up/Down arrows)  
- Controllable **rain intensity** (+ / - keys)  
- A cute hand-drawn-style house with roof, door, and window  

Perfect for learning classic OpenGL animation techniques, immediate-mode rendering, or just enjoying some cozy pixel rain vibes.

# Controls

| Key                  | Action                              |
|----------------------|-------------------------------------|
| `←` `→`              | Blow rain left / right (wind)       |
| `R`                  | Reset wind                          |
| `W` / `S`            | Increase / decrease wind strength   |
| `↑` `↓`              | Make it day / night                 |
| `+` / `-`            | Faster / slower rain                |

# Tech Stack

- **Python 3**
- `PyOpenGL` + `glfw` (no external shaders or modern OpenGL needed)
- Pure immediate-mode rendering (`glBegin/glEnd`) — old-school and charming!
